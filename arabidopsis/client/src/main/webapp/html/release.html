Arabidopsis Release Notes

The final Arabidopsis thaliana information network contains:

    Datasets used for integration

    • Gene- Protein [Ensembl]
      • Arabidopsis_thaliana.TAIR10.36.gff3
      • Arabidopsis_thaliana.TAIR10.pep.all.fa
      • Ensembl - UniprotKB Mapping
    • Gene-SNP-Phenotype [Ensembl]
      • Arabidopsis GWAS data for 92 phenotypes.
      • Phenotypes are grouped into the four categories: flowering, developmental, defense and ionomics.
      • For more information see Atwell et al, 2010, Nature and supplementary information
      • Data was downloaded from Ensembl Biomart
      • SNP-Phenotype relations (27,466 relations) of significant SNPs (p-value<0.0001) linked to 92 phenotypes; on average 300 SNPs per phenotype.
      • SNP-Gene relations are based on genes in close proximity to SNPs <1000bp (13,331 relations)
      • #SNP=11,996 | #Gene=8,241 | #Phenotype=92
      • To see it live, go to KnetMiner Arabidopsis and search for dormancy, then switch to Map View.
    • Gene-GO [GOA]
      • Include all evidence types
      • GAF 2.0
      • Ondex GAF parser requires additional mapping file of feature type -> Ondex Concept Class
    • Gene-Gene Interactions [BioGRID]
      • TAB
      • Version 3.4.151
      • Concepts: 9212 [Gene] | 2359 [Publication]
      • Relations: 303 [genetic] | 37083 [physical] | 19295 [pub_in]
      • filter out High Throughput interactions for gene AT5G03240
    • Gene-Phenotype [TAIR]
      • TAB
