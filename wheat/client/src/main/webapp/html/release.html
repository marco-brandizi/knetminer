Triticum aestivum

Wheat Release Notes

The Wheat (Triticum aestivum) information network is TGACv1.0 and contains:

    Datasets used for integration

    • Gene-Protein [Ensembl]
      • Triticum_aestivum.TGACv1.36.gff3
      • Triticum_aestivum.TGACv1.pep.all.fa
      • Ensembl-UniprotKB-SP Mapping File from Ensembl BioMart
    • Protein-Domain [Ensembl]
      • InterPro Protein Domains from Ensembl BioMart
    • Gene-SNP-Phenotype [T3]
      • Wheat GWAS data for 49 phenotypes
      • Data was provided by Triticeae Toolbox
    • Expression [EBI-GXA]
      • Data was downloaded from Gene Expression Atlas
      • E-MTAB-4222 experiment
        • 'NIL38; Fusarium graminearum' at '24 hour' vs 'NIL38; mock' at '24 hour'
      • E-MTAB-1729 experiment
        • 'Fusarium graminearum' vs 'mock' in 'CM-82036 resistant parent line' at '30 hour'
    • Protein-Publication [UniProtKB]
      • Ta UniprotKB-SwissProt
    • Gene-TO [Text Mining]
      • Gene-TO sentence based co-occurrences in PubMed abstracts
    • Publications [PubMed]
      • XML
      • filter publications that are unconnected, i.e. not linked via citation or text-mining
    • Homology
      • Wheat-Arabidopsis homologs from Ensembl BioMart
      • Sequence alignments to other Plants
        • Protein@Wheat-Protein@Plants::UniProtKB-SP [Smith-Waterman]
    • References
      • Arabidopsis (TAIR) see Release notes for Arabidopsis
