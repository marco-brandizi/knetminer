Maize Release Notes

The Maize (AGPv4) information network contains:

    Datasets used for integration

    • Gene-Protein [Ensembl]
      • Zea_mays.AGPv4.36.gff3.gz
      • Zea_mays.AGPv4.pep.all.fa.gz
      • Mapping File from Ensembl BioMart
      • Ensembl - UniprotKB Mapping
      • Gene Names from MaizeGDB
      • AGPv3 to AGPv4 ID Mapping from Gramene
    • Protein-Domain [Ensembl]
      • InterProt Protein Domains from Ensembl BioMart
    • Protein-Publication [UniProtKB]
      • Zm Proteome xml
    • Publications [PubMed]
      • XML
      • filter publications that are unconnected, i.e. not linked via citation or text-mining
    • Gene-TO [Text-Mining]
      • Gene-TO sentence based co-occurrences in PubMed abstracts
    • Homology
      • Maize-Arabidopsis homologs from Ensembl BioMart
      • Sequence alignments to other Plants
        • Protein@Maize-Protein@Plants::UniProtKB-SP [Smith-Waterman]
    • References
      • Arabidopsis (TAIR) see Release notes for Arabidopsis
